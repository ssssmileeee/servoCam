# servoCam

Arduino-скетч для управления самодельной моторизированной (выдвижной/откидной) камерой заднего вида в легковом автомобиле. Камера открывается при включении заднего хода и закрывается при его выключении.

## Состав проекта

- `servoCam.ino` — единственный скетч, вся логика в нём.
- `README.MD` — описание проекта на английском, схема подключения в `img/layout.png`, фото в `img/cam1.jpg`.
- Зависимость: библиотека [ServoSmooth](https://alexgyver.ru/servosmooth/) (плавное движение сервопривода с настройкой скорости и ускорения).

## Железная часть

- Контроллер: Arduino Pro Micro (подойдёт любой контроллер с хотя бы одним аналоговым входом).
- Питание Arduino: от бортовой сети 12V через понижающий DC-DC-преобразователь.
- **D2** — сигнальный провод сервопривода **MG90S**, который открывает/закрывает камеру.
- **A0** — вход сигнала заднего хода: питание фонарей заднего хода (12V) заведено через резистивный делитель **3k / 1k** (верхнее плечо 3k, нижнее 1k, коэффициент 1/4 — безопасно читает уровни до ~20V).

## Логика работы (servoCam.ino)

1. В `loop()` постоянно читается `analogRead(A0)`; если значение ≥ `reverseSignalMinVoltage` (450) — задний ход включён.
2. Открытие камеры происходит с задержкой `openCamDelayMs` (500 мс) — защита от коротких ложных срабатываний (например, при переключении передач).
3. Закрытие происходит сразу при пропадании сигнала.
4. Сервопривод двигается плавно: `setSpeed(500)`, `setAccel(1.2)`.

## Ключевые константы для подстройки под конкретную установку

| Константа | Значение | Смысл |
|---|---|---|
| `reverseSignalPin` | A0 | Вход сигнала фонарей заднего хода |
| `servoPin` | 2 (D2) | Сигнальный пин сервопривода |
| `maxServoAngle` | 127 | Максимальный угол сервопривода |
| `closedAngle` | 127 | Угол в закрытом состоянии |
| `openedAngle` | 20 | Угол в открытом состоянии |
| `reverseSignalMinVoltage` | 450 | Порог АЦП для детекта заднего хода (~2.2V на пине ≈ ~8.8V на входе делителя) |
| `openCamDelayMs` | 500 | Задержка перед открытием, мс |
